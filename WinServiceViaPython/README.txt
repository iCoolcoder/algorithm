This program check the status of Bit Locker client service. If it is enabled, stop the service.

This program implemented by Python.

python3.3

install service:
python WinSeriveViaPython.py install
start service:
python WinSeriveViaPython.py start
restart service:
python WinSeriveViaPython.py restart
stop service:
python WinSeriveViaPython.py stop
remove service:
python WinSeriveViaPython.py remove
